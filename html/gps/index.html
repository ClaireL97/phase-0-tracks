Q:What is the purpose of HTML5 semantic tags? Which ones do you see as the most useful?
A: I think the purpose of the HTML 5 semantic tags is to describe the meaning of exactly what is inside the tags and keep it uniform so the developer and the computer don't have to work harder to find out what is inside the tag. I think the most usesful ones are the header, footer, nav and main. Header and footer because thats where you can link to other pages or put contact information and the nav because thats how you link to other sites and pages. Finally the main because thats where the most of the sites content goes for a paragraph or main purpose of the site.

Q:What is the order the browser uses to display elements on the page?
A: The browser displays elements on the page from top to bottom. Depending on the order of the tags in the body, it goes from top to bottom displaying the contents of each tab.
